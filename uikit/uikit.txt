uikit--
здесь живут:
- папки для каждой из страниц UiKit. В каждой из них находятся:
	- папки для каждого блока, который используется на этой странице. В каждой папка блока:
		- файл block.pug
		- block.scss
		- block.js
		- картинки, который используются в этом блоке

	- (файлы самой страницы, в которые импортируются файлы блоков) index.pug
	- index.scss
	- index.js

	- служебные файлы и папки parcel.
